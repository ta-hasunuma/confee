# プロダクト概要

## プロダクト名

**Confee** — Your Tech Conference Partner

## 概要

Confee は、日本の技術カンファレンス・勉強会・LT会の情報を自然言語で検索・推薦する AI エージェントアプリケーション。connpass API v2 を通じてイベント情報を取得し、ユーザーのスキルや関心に基づいておすすめのカンファレンスを提案する。

## コア機能

- **自然言語によるカンファレンス検索**: 「TypeScriptのカンファレンスある？」のようなクエリで connpass イベントを検索
- **インテリジェントな推薦**: 曖昧なクエリ（「面白そうなカンファレンス見つけて」）に対しても推薦スコア付きで回答
- **構造化されたイベント情報表示**: イベント名、日時、場所、参加状況、推薦スコア、申込リンクを Markdown 形式で提示
- **おすすめプロンプト表示**: 初回アクセス時にクリッカブルなサンプルクエリを表示し、ユーザーの利用を促進
- **チャットインターフェース**: ログイン不要で利用可能な会話型 UI

## ターゲットユースケース

- 技術カンファレンスや勉強会を探しているエンジニア
- 自分のスキルや興味に合ったイベントを見つけたいユーザー
- 開催間近のイベントを素早くチェックしたいユーザー

## 主要な価値提案

- **自然言語での検索**: キーワードや条件を組み合わせた検索フォームではなく、自然な日本語で問い合わせ可能
- **AI による推薦**: 単なる検索結果の羅列ではなく、推薦スコアと理由付きで提案
- **即座に利用可能**: ログインやアカウント登録不要。アクセスしてすぐに使える

## 現在のステータス

MVP（Minimum Viable Product）として全 6 タスクの実装が完了。AWS 環境にデプロイ済み。

## 外部データソース

- **connpass API v2**: 日本最大級の技術イベントプラットフォーム connpass からイベント情報を取得
  - API キー未設定時はモックデータで動作可能
